# Demo Images

These images are generated from the synthetic demo project in `examples/demo_project/` and are intended for direct use in the main README.

- `demo_te_summary.png`: TE transport summary plot generated from synthetic processed CSV files.
- `demo_xrd_pdf_overlay.png`: normalized XRD stack with a synthetic PDF-card stick overlay.
- `demo_room_temp_dual_axis.png`: optional room-temperature Seebeck/electrical-conductivity comparison.

The source data are synthetic and included only to verify plotting behavior without publishing private experimental data.
