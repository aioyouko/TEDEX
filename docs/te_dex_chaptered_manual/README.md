# TEDEX Chaptered Manual

This folder contains a chapter-based LaTeX manual for the TEDEX workspace.

Build entry point:

```bash
main.tex
```

Folder layout:

```text
docs/te_dex_chaptered_manual/
├── main.tex
├── chapters/
│   ├── 00_outline.tex
│   ├── 01_introduction.tex
│   ├── 02_workspace_architecture.tex
│   ├── 03_raw_te_analysis.tex
│   ├── 04_spb_fitting.tex
│   ├── 05_plotting_and_figures.tex
│   ├── 06_xrd_analysis.tex
│   ├── 07_assessment_and_prediction.tex
│   └── 08_reproducible_operations.tex
└── figures/
    └── README.md
```

The SPB fitting chapter is the first fully drafted chapter. Other chapters are
structured outlines with TODO slots for commands, terminal output, and figures.
