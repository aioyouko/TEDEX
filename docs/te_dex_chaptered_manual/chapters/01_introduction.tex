\chapter{Introduction}

\section{Purpose}

TEDEX is a structured workspace for thermoelectric materials analysis. It is
designed to connect raw laboratory measurements, sample metadata, Python
analysis scripts, generated figures, and written reports without forcing every
experiment into a rigid database-first workflow.

\section{Planned Content}

\begin{itemize}[leftmargin=*]
  \item Scientific motivation: why reproducible TE analysis matters for
  composition screening and optimization.
  \item Target users: lab members processing CHY batches, preparing figures, or
  comparing measured trends against SPB models.
  \item Design principles: runnable root scripts, reusable source modules,
  explicit metadata ledgers, and separated generated outputs.
  \item Short comparison to notebook-only workflows.
\end{itemize}
