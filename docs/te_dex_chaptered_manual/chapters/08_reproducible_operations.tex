\chapter{Reproducible Operations}

\section{End-to-End Protocol}

\begin{enumerate}[leftmargin=*]
  \item Place original instrument files under \code{data/raw/<batch_id>/}.
  \item Run metadata synchronization.
  \item Fill missing density, heat-capacity, composition, and notes fields.
  \item Run raw TE analysis.
  \item Generate TE and XRD figures.
  \item Run optional SPB, assessment, and prediction workflows.
  \item Archive the exact commands, output paths, and figure files used in any
  report.
\end{enumerate}
